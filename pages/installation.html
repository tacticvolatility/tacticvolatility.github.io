Installation

Using Docker Image

The contents of the Dockerfile are as follows: 

FROM python:latest
    
    ADD . /celiatool
    
    WORKDIR /celiatool
    
    # Install native libraries, required for numpy
    RUN pip --no-cache
    
    # Upgrade pip
    RUN pip install --upgrade pip
    
    # packages that we need
    RUN pip install numpy
    RUN pip install pandas
    RUN pip install jupyter
    RUN pip install sklearn
    
    
    CMD ["main.py"]
    
    ENTRYPOINT python3 main.py ./config.json

The image is availble as celiatool.tar file and can be accessed at: http://drive.google.com/file/d/17P7u0z5uQpVDAGns3xla5kBhBxYryfAS/view?usp=sharing


Once the image is dowloaded, to load the docker image you'll have use the command:

docker load -input celiatool.tar

To check if the image has been downloaded successfully you check it with a simple command:

 docker images
