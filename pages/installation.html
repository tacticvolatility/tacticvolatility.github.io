Installation

Using Docker Image

Docker is a containerization platform. It helps to package units of code with their dependencies so that they can run in isolation. And a docker image is basically a snapshot of the dockerized code that is created when the docker build command is used.


Basic Requirements to run the docker image:

  1. Install Docker
  2. Create a docker id
  3. Log in with the docker id on Dockerhub


The contents of the Dockerfile are as follows: 

FROM python:latest
    
    ADD . /celiatool
    
    WORKDIR /celiatool
    
    # Install native libraries, required for numpy
    RUN pip --no-cache
    
    # Upgrade pip
    RUN pip install --upgrade pip
    
    # packages that we need
    RUN pip install numpy
    RUN pip install pandas
    RUN pip install jupyter
    RUN pip install sklearn
    
    
    CMD ["main.py"]
    
    ENTRYPOINT python3 main.py ./config.json

The image is availble as celiatool.tar file and can be accessed at:

  • Docker Image Download


    Once the image is dowloaded, to load the docker image you'll have use the command:

    docker load -input celiatool.tar

    To check if the image has been downloaded successfully you check it with a simple command:

     docker images
